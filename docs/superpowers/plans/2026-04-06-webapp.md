# Ferrite Webapp Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a browser-based webapp that lets you run and debug all Ferrite features (sprite editing, state machine authoring, pet simulation, bundle import/export) in a wasm32 eframe app, sharing all egui panel code with the desktop.

**Architecture:** Extract shared egui panels into a new `crates/ferrite-egui` crate, add portable types (bundle, editor_state) to `ferrite-core`, then build `crates/ferrite-webapp` as an eframe-wasm app that reuses those panels. Platform I/O (filesystem vs. browser storage) is abstracted via `SmStorage` and `SheetLoader` traits defined in `ferrite-egui`.

**Tech Stack:** Rust 2024, eframe 0.33 (glow backend for wasm), ferrite-core (portable logic), ferrite-egui (shared panels), wasm-bindgen, web-sys, rfd (async, wasm-compatible), trunk (build tool), Playwright (E2E tests).

- [ ] **Step 2: Write `crates/ferrite-egui/src/sm_storage.rs`**

```rust
/// Platform-agnostic interface for reading and writing state machine source files.
/// Desktop: wraps `SmGallery` + `config_dir` filesystem operations.
/// Web: in-memory `HashMap<String, String>`.
pub trait SmStorage: Send + Sync {
    /// Returns all valid SM names (excluding drafts).
    fn list_names(&self) -> Vec<String>;
    /// Returns the TOML source for the named SM, or `None` if not found.
    fn load(&self, name: &str) -> Option<String>;
    /// Saves source under `name`. Returns an error string on failure.
    fn save(&self, name: &str, source: &str) -> Result<(), String>;
    /// Deletes the SM named `name`. Returns an error string on failure.
    fn delete(&self, name: &str) -> Result<(), String>;
}
```

```rust
//! Desktop wrapper: provides `DesktopSmStorage` (filesystem-backed) and
//! re-exports the platform-agnostic SM editor from `ferrite-egui`.

pub use ferrite_egui::sm_editor::{
    render_sm_panel, SmEditorFromApp, SmEditorFromUi, SmEditorViewport, VarSnapshot,
};

use ferrite_egui::sm_storage::SmStorage;
use crate::sprite::sm_gallery::SmGallery;
use std::path::PathBuf;

pub struct DesktopSmStorage {
    pub config_dir: PathBuf,
}

impl SmStorage for DesktopSmStorage {
    fn list_names(&self) -> Vec<String> {
        SmGallery::load(&self.config_dir)
            .valid_names()
            .into_iter()
            .map(|s| s.to_string())
            .collect()
    }

    fn load(&self, name: &str) -> Option<String> {
        // `source()` returns the raw TOML for a valid SM.
        SmGallery::load(&self.config_dir).source(name).map(|s| s.to_string())
    }

    fn save(&self, name: &str, source: &str) -> Result<(), String> {
        // SmGallery::save returns Ok(bool) — true=valid, false=draft. We discard the bool.
        let mut gallery = SmGallery::load(&self.config_dir);
        gallery.save(name, source).map(|_| ()).map_err(|e| e.to_string())
    }

    fn delete(&self, name: &str) -> Result<(), String> {
        let mut gallery = SmGallery::load(&self.config_dir);
        gallery.delete(name).map_err(|e| e.to_string())
    }
}

/// Desktop constructor: creates `SmEditorViewport` with filesystem-backed storage.
pub fn new_desktop_sm_editor(dark_mode: bool, config_dir: PathBuf) -> SmEditorViewport {
    let storage = Box::new(DesktopSmStorage { config_dir });
    SmEditorViewport::new(dark_mode, storage)
}
```

pub fn open_app_window(
    ctx: &egui::Context,
    state: Arc<Mutex<AppWindowState>>,
    tx: Sender<AppEvent>,
    window_gen: u64,
    close_flag: Arc<AtomicBool>,
) {
    let viewport_id = egui::ViewportId::from_hash_of(format!("app_window_{window_gen}"));
    let viewport_builder = egui::ViewportBuilder::default()
        .with_title("Ferrite")
        .with_inner_size([1000.0, 640.0]);

    ctx.show_viewport_deferred(viewport_id, viewport_builder, move |ctx, _vp_class| {
        if ctx.input(|i| i.viewport().close_requested()) {
            close_flag.store(true, Ordering::Relaxed);
            return;
        }
        let mut s = match state.lock() { Ok(g) => g, Err(_) => return };
        if s.should_close {
            close_flag.store(true, Ordering::Relaxed);
            return;
        }
        // Propagate config changes to the desktop app event loop
        if s.config_state.config_dirty {
            s.config_state.config_dirty = false;
            tx.send(AppEvent::ConfigChanged(s.config_state.config.clone())).ok();
        }
        ferrite_egui::app_window::render_full_window(ctx, &mut s);
    });
}

use ferrite_core::config::schema::Config;

const STORAGE_KEY: &str = "ferrite_config";

/// Load config from localStorage. Returns `Config::default()` if absent or invalid.
pub fn load() -> Config {
    let window = web_sys::window().expect("no window");
    let storage = window.local_storage().ok().flatten().expect("no localStorage");
    let toml_str = match storage.get_item(STORAGE_KEY) {
        Ok(Some(s)) => s,
        _ => return Config::default(),
    };
    toml::from_str(&toml_str).unwrap_or_default()
}

/// Persist config to localStorage.
pub fn save(config: &Config) {
    let Ok(toml_str) = toml::to_string_pretty(config) else { return };
    let window = web_sys::window().expect("no window");
    let storage = window.local_storage().ok().flatten().expect("no localStorage");
    storage.set_item(STORAGE_KEY, &toml_str).ok();
}

    last_tick: f64,  // performance.now() ms

impl WebApp {
    pub fn new() -> Self {
        let config = crate::config_store::load();
        let storage = Box::new(crate::web_storage::WebSmStorage::new());
        let loader = Box::new(crate::web_storage::WebSheetLoader::new());
        let gallery = crate::web_storage::build_gallery();

    fn update(&mut self, ctx: &egui::Context, _frame: &mut eframe::Frame) {
        // Persist config changes
        if self.state.config_state.config_dirty {
            self.state.config_state.config_dirty = false;
            crate::config_store::save(&self.state.config_state.config);
        }

        // Tick pet simulation
        let now = js_sys::Date::now();
        let delta_ms = if self.last_tick == 0.0 { 16.0 } else { (now - self.last_tick).min(100.0) };
        self.last_tick = now;

/// In-memory SM storage for the webapp.
#[derive(Clone)]
pub struct WebSmStorage {
    inner: Arc<Mutex<HashMap<String, String>>>,
}

impl WebSmStorage {
    pub fn new() -> Self {
        Self { inner: Arc::new(Mutex::new(HashMap::new())) }
    }
}

impl SmStorage for WebSmStorage {
    fn list_names(&self) -> Vec<String> {
        self.inner.lock().unwrap().keys().cloned().collect()
    }
    fn load(&self, name: &str) -> Option<String> {
        self.inner.lock().unwrap().get(name).cloned()
    }
    fn save(&self, name: &str, source: &str) -> Result<(), String> {
        self.inner.lock().unwrap().insert(name.to_string(), source.to_string());
        Ok(())
    }
    fn delete(&self, name: &str) -> Result<(), String> {
        self.inner.lock().unwrap().remove(name);
        Ok(())
    }
}

/// In-memory sheet loader backed by bytes loaded from network or bundle import.
pub struct WebSheetLoader {
    sheets: Arc<Mutex<HashMap<String, (Vec<u8>, Vec<u8>)>>>, // path → (json, png)
}

impl WebSheetLoader {
    pub fn new() -> Self {
        Self { sheets: Arc::new(Mutex::new(HashMap::new())) }
    }

    /// Register a sheet (called after bundle import or asset fetch).
    pub fn register(&self, path: String, json: Vec<u8>, png: Vec<u8>) {
        self.sheets.lock().unwrap().insert(path, (json, png));
    }
}

impl SheetLoader for WebSheetLoader {
    fn load_sheet(&self, path: &str) -> anyhow::Result<SpriteSheet> {
        if let Some(stem) = path.strip_prefix("embedded://") {
            return load_embedded_sheet(stem);
        }
        let map = self.sheets.lock().unwrap();
        let (json, png) = map.get(path)
            .ok_or_else(|| anyhow::anyhow!("sheet not found: {path}"))?;
        let image = image::load_from_memory_with_format(png, image::ImageFormat::Png)?.into_rgba8();
        SpriteSheet::from_json_and_image(json, image)
    }
}

/// Load an embedded sheet by its stem name.
/// The esheep assets (esheep.json + esheep.png) must be copied into the webapp's
/// static assets by Trunk (configured in Trunk.toml with a `[[assets]]` copy rule).
/// They are served at `/assets/{stem}.json` and `/assets/{stem}.png`.
/// For wasm, we use `gloo-net` or the `include_bytes!` approach.
/// Simplest: include them at compile time if they are in the ferrite-webapp crate directory.
fn load_embedded_sheet(stem: &str) -> anyhow::Result<SpriteSheet> {
    // Compile-time embedding of known assets:
    match stem {
        "esheep" => {
            let json: &[u8] = include_bytes!("../assets/esheep.json");
            let png: &[u8] = include_bytes!("../assets/esheep.png");
            let image = image::load_from_memory_with_format(png, image::ImageFormat::Png)?.into_rgba8();
            ferrite_core::sprite::sheet::SpriteSheet::from_json_and_image(json, image)
        }
        _ => anyhow::bail!("unknown embedded sheet: {stem}"),
    }
}

use eframe::egui::{self, Color32, Pos2, Rect, Stroke, Vec2};
use ferrite_core::{
    config::schema::{Config, PetConfig},
    sprite::{
        animation::AnimationState,
        sheet::SpriteSheet,
        sm_runner::SMRunner,
    },
};
use std::collections::HashMap;

const FLOOR_HEIGHT: f32 = 20.0;

/// A draggable surface box inside the simulation.
pub struct SurfaceBox {
    pub rect: Rect,
    pub dragging: bool,
    pub drag_offset: Vec2,
}

/// Runtime state for one pet in the simulation.
struct SimPet {
    cfg: PetConfig,
    sheet: SpriteSheet,
    anim: AnimationState,
    runner: SMRunner,
    x: f32,
    y: f32,
    texture: Option<egui::TextureHandle>,
    texture_dirty: bool,
}

pub struct SimulationState {
    pets: Vec<SimPet>,
    surfaces: Vec<SurfaceBox>,
}

impl SimulationState {
    pub fn new(config: Config) -> Self {
        let mut pets = Vec::new();
        for cfg in config.pets {
            if let Ok(sheet) = crate::web_storage::load_sheet_for_config(&cfg.sheet_path) {
                let runner = SMRunner::new(
                    ferrite_core::sprite::sm_runner::load_default_sm(),
                    cfg.walk_speed,
                );
                let anim = AnimationState::new("fall".to_string());
                pets.push(SimPet { cfg, sheet, anim, runner, x: 100.0, y: 0.0, texture: None, texture_dirty: true });
            }
        }
        Self { pets, surfaces: Vec::new() }
    }

    pub fn add_surface(&mut self, rect: Rect) {
        self.surfaces.push(SurfaceBox { rect, dragging: false, drag_offset: Vec2::ZERO });
    }

    pub fn render(&mut self, ctx: &egui::Context) {
        egui::CentralPanel::default().show(ctx, |ui| {
            let panel_rect = ui.available_rect_before_wrap();
            let floor_y = panel_rect.max.y - FLOOR_HEIGHT;

            // Draw background
            let painter = ui.painter();
            painter.rect_filled(panel_rect, 0.0, Color32::from_rgb(30, 30, 50));

            // Draw floor
            let floor_rect = Rect::from_min_max(
                Pos2::new(panel_rect.min.x, floor_y),
                panel_rect.max,
            );
            painter.rect_filled(floor_rect, 0.0, Color32::from_rgb(70, 80, 100));
            painter.text(
                Pos2::new(panel_rect.min.x + 4.0, floor_y + 4.0),
                egui::Align2::LEFT_TOP,
                "floor",
                egui::FontId::proportional(11.0),
                Color32::from_rgb(120, 140, 180),
            );

            // Draw and handle surface boxes
            for surf in &mut self.surfaces {
                painter.rect_stroke(surf.rect, 4.0, Stroke::new(1.5, Color32::from_rgb(100, 180, 120)), egui::StrokeKind::Outside);
                painter.rect_filled(surf.rect, 4.0, Color32::from_rgba_premultiplied(100, 180, 120, 30));
            }

            // Draw pets
            for pet in &mut self.pets {
                let frame_w = pet.sheet.frames.first().map(|f| f.w).unwrap_or(32);
                let frame_h = pet.sheet.frames.first().map(|f| f.h).unwrap_or(32);
                let scaled_w = frame_w as f32 * pet.cfg.scale;
                let scaled_h = frame_h as f32 * pet.cfg.scale;
                let pet_rect = Rect::from_min_size(
                    Pos2::new(pet.x, pet.y),
                    Vec2::new(scaled_w, scaled_h),
                );
                // Upload texture if dirty
                if pet.texture_dirty || pet.texture.is_none() {
                    if let Some(frame) = pet.anim.current_frame_index().and_then(|i| pet.sheet.frames.get(i)) {
                        let sub = image::imageops::crop_imm(
                            &pet.sheet.image,
                            frame.x, frame.y, frame.w, frame.h,
                        ).to_image();
                        let pixels: Vec<egui::Color32> = sub.pixels()
                            .map(|p| egui::Color32::from_rgba_unmultiplied(p[0], p[1], p[2], p[3]))
                            .collect();
                        let tex = ctx.load_texture(
                            format!("pet_{}", pet.cfg.id),
                            egui::ColorImage { size: [frame.w as usize, frame.h as usize], pixels },
                            egui::TextureOptions::NEAREST,
                        );
                        pet.texture = Some(tex);
                        pet.texture_dirty = false;
                    }
                }
                if let Some(ref tex) = pet.texture {
                    painter.image(tex.id(), pet_rect, Rect::from_min_max(Pos2::ZERO, Pos2::new(1.0, 1.0)), Color32::WHITE);
                } else {
                    painter.rect_filled(pet_rect, 2.0, Color32::from_rgb(200, 100, 100));
                }
            }

            // "+ Add Surface" button
            if ui.button("+ Add Surface").clicked() {
                self.add_surface(Rect::from_min_size(
                    Pos2::new(panel_rect.center().x - 60.0, floor_y - 60.0),
                    Vec2::new(120.0, 20.0),
                ));
            }
        });
    }
}

use wasm_bindgen::prelude::*;
use wasm_bindgen_futures::JsFuture;
use web_sys::{Blob, BlobPropertyBag, Url};
use js_sys::{Array, Uint8Array};
use ferrite_core::bundle;

/// Trigger a browser file download with the given bytes and filename.
pub fn download_bytes(bytes: &[u8], filename: &str, mime: &str) {
    let array = Uint8Array::from(bytes);
    let parts = Array::new();
    parts.push(&array.buffer());
    let mut props = BlobPropertyBag::new();
    props.type_(mime);
    let blob = Blob::new_with_u8_array_sequence_and_options(&parts, &props)
        .expect("Blob creation failed");
    let url = Url::create_object_url_with_blob(&blob).expect("createObjectURL failed");
    let document = web_sys::window().unwrap().document().unwrap();
    let a = document
        .create_element("a").unwrap()
        .dyn_into::<web_sys::HtmlAnchorElement>().unwrap();
    a.set_href(&url);
    a.set_download(filename);
    a.click();
    Url::revoke_object_url(&url).ok();
}

/// Import a .petbundle from raw bytes. Returns `BundleContents` on success.
pub fn import_bundle(data: &[u8]) -> Result<bundle::BundleContents, String> {
    bundle::import(data)
}

/// Export the current sprite + SM as a .petbundle and trigger download.
pub fn export_bundle(
    bundle_name: &str,
    sprite_json: &str,
    sprite_png: &[u8],
    sm_source: Option<&str>,
) {
    match bundle::export(bundle_name, None, sprite_json, sprite_png, sm_source, None) {
        Ok(bytes) => download_bytes(&bytes, &format!("{bundle_name}.petbundle"), "application/zip"),
        Err(e) => log::error!("bundle export failed: {e}"),
    }
}

/// Open a file picker and read a .petbundle. Returns the bytes via an async callback.
pub async fn pick_and_read_bundle() -> Option<Vec<u8>> {
    let handle = rfd::AsyncFileDialog::new()
        .add_filter("Pet Bundle", &["petbundle"])
        .pick_file()
        .await?;
    Some(handle.read().await)
}
